Шаги для запуска проект
1. Установить psycopg2: pip install psycopg2
2. Установить пакеты для работы с http: pip install fastapi uvicorn pydantic bcrypt
3. Запустить команду:  uvicorn main:app --reload --port 8989
4. Импортировать файл который скину в группу в postman
5. Протестировать роуты
6. Чтобы перейти к swagger: 127.0.0.1:8989/docs
7. Установить пакеты для работы с JWT: pip install fastapi uvicorn PyJWT passlib python-multipart


------------------------------------------------
Описание файлов:
- db.py - хранит логику работы с базой данных
- models.py - хранит список наших сущностей (классов)
- main.py - точка запуска приложения
